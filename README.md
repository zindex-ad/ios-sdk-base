# ZindexBase iOS Framework

这是ZindexBase的发布版本，包含编译好的framework供第三方使用。

## 安装

### CocoaPods

```ruby
pod 'ZindexYLH', :git => 'https://github.com/zindex-ad/ios-sdk-base.git', :tag => '1.0.0'
```

### Swift Package Manager

```swift
dependencies: [
    .package(url: "https://github.com/zindex-ad/ios-sdk-base.git", from: "1.0.0")
]
```

### 手动安装

1. 下载对应版本的framework
2. 将ZindexBase.framework拖入项目
3. 添加必要的系统框架依赖

## 版本历史
